Command Prompt (CMD) Commands

The Command Prompt (CMD) is a command-line interpreter application available in most Windows operating systems. It allows users to execute commands to perform various tasks such as file management, network configuration, and system troubleshooting.

Commonly Used CMD Commands

  • dir: Lists the files and directories in the current directory.
    Example: dir
  • cd: Changes the current directory.
    Example: cd C:\Users
  • copy: Copies one or more files to another location.
    Example: copy file.txt D:\Backup
  • del: Deletes one or more files.
    Example: del file.txt
  • ipconfig: Displays network configuration details.
    Example: ipconfig /all
  • ping: Tests connectivity to a network host.
    Example: ping google.com
  • netstat: Displays active network connections and listening ports.
    Example: netstat -an
  • tasklist: Lists all running processes.
    Example: tasklist
  • shutdown: Shuts down or restarts the computer.
    Example: shutdown /s /t 0 (shutdown immediately)
  • cls: Clears the command prompt screen.
    Example: cls

Tips for Using CMD Effectively

  • Use help followed by a command name to get detailed information about that command. Example: help dir
  • Use tab completion to quickly complete file and directory names.
  • Run CMD as administrator to execute commands that require elevated privileges.
  • Use command chaining with && or || to execute multiple commands conditionally.
  • Redirect output to files using > or >> operators.
